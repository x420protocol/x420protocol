x420 Protocol

Optimized micro-payment middleware for Express.js

Single-request payment authorization with cryptographic security. Stateless, fast, and payment-provider agnostic.

Why x420?

Built for performance, security, and developer experience

Single-Request Access

Consolidates price discovery and payment authorization into one request. Minimal latency, maximum efficiency.

Cryptographically Secure

Every access secured by unique, atomic, on-chain transactions. Stateless verification with signature-based authorization.

Price Mismatch Failsafe

Automatic price validation prevents stale cache exploitation. 402 responses trigger immediate price refresh.

Provider Agnostic

Abstract PaymentProvider interface works with any blockchain, payment processor, or custom authorization scheme.

Fully Stateless

No server-side sessions or pre-paid tokens. Each request carries complete cryptographic proof.

TypeScript First

Complete type definitions, IntelliSense support, and compile-time safety for your payment flows.

How It Works

Phase 1

Price Discovery

Clients retrieve current prices for protected resources via a dedicated endpoint.

1
Proactive Discovery

Client requests /x420-prices and caches the response locally

2
Reactive Discovery

On 402 response, client invalidates cache and refreshes prices

Installation

Implement Payment Provider

import { PaymentProvider, PaymentAuthorization, PaymentVerificationResult } from 'x420-middleware';

class MyPaymentProvider implements PaymentProvider {
  async verifyPayment(auth: PaymentAuthorization): Promise {
    const isValid = await this.verifySignature(auth.signature, auth.wallet);
    
    if (!isValid) {
      return { valid: false, error: 'Invalid signature' };
    }
    
    return {
      valid: true,
      wallet: auth.wallet,
      amount: auth.amount,
      asset: auth.asset
    };
  }

  async settlePayment(auth: PaymentAuthorization): Promise {
    await this.submitToBlockchain(auth);
  }

  private async verifySignature(signature: string, wallet: string): Promise {
    return true;
  }

  private async submitToBlockchain(auth: PaymentAuthorization): Promise {
    console.log('Settling payment:', auth);
  }
}

Configure Middleware

import express from 'express';
import { x420 } from 'x420-middleware';

const app = express();

const middleware = x420({
  paymentProvider: new MyPaymentProvider(),
  priceMap: {
    '/api/premium-data': { price: '100', asset: 'USDC' },
    '/api/exclusive-content': { price: '50', asset: 'USDC' }
  },
  priceDiscoveryPath: '/x420-prices',
  paymentHeader: 'x-payment'
});

app.use(middleware);

app.get('/api/premium-data', (req, res) => {
  res.json({ 
    wallet: req.wallet,
    data: 'This is premium content' 
  });
});

app.listen(3000, () => {
  console.log('Server running on port 3000');
});
